# Cypress NextJS Test 🚦

Fullstack NextJS app, just to explore Cypress library features on NextJS client-side & server-side modes.

## Tools 🛠

- NodeJS v20.13.1
- NextJS v15.5.3

## How to start

```bash
npm i
npm run dev
```
